Obsidian Markdown Cheat-Sheet Reference

Headers & Structural Formatting

  • # Header 1 Title Level 1
  • ## Header 2 Section Title
  • ### Header 3 Subtitle Element
  • --- Horizontal Rule

Highlighting & Emphasis styles

  • **bold text** bold text
  • *italicized* italic text
  • ~~strikethrough~~ deleted text
  • ==highlight passage== Passage Highlighting

Interactivity List Elements

  • - [ ] Incomplete task Checkbox empty
  • - [x] Finished task Checked state
  • * bullet item Standard lists
  • 1. ordered lists Decimal count list

Advanced Components Code & Quotes

  • `inline code` Code chips inline
  • > Quote text block Styled quote banner
  • ::Keyboard shortcut:: kbd capsule chips
  • | Table Header | Auto align table lists

Pro Tip: Checklist actions are dynamic. Clicking a checkbox inside the Live Preview pane modifies the source Markdown document automatically and triggers real-time saving.
